Bem-vindo ao guia da Folha Digital Sólides

Uma leitura pensada para negócio, produto e diretoria: entenda o que estamos entregando, por que importa e como as seis etapas do wizard transformam caos de planilhas em jornada guiada — sem pedir que você decore sigla de sistema.

Tom leve, conteúdo sério Pronto para apresentar

Se você veio parar aqui por engano procurando o app: ele roda no Vite com rotas tipo #/dashboard. Esta pasta é só documentação estática — tipo “slide que não depende do PowerPoint travar”.

Três razões para a diretoria apostar nesta abordagem

🧭

Rumo claro

Cada etapa tem começo, meio e “o que falta para fechar”. Menos dependência de reunião informal para saber o status do projeto de folha.

🛡️

Risco menor

Documentos e decisões ficam associados ao fluxo. Na hora do pente fino — ou de uma auditoria — não precisamos caçar e-mail de seis meses atrás.

🤝

Alinhamento RH × TI × consultoria

O mesmo roteiro serve para explicar ao cliente externo, ao time interno e à mesa executiva: menos “tradutor humano” no meio.

Comece por aqui

O que vamos entregar Texto completo para reunião de diretoria: capítulos, valor, glossário e próximos passos. Wizard em português claro Visão geral das seis etapas + como navegar no app. Jornada do cliente Fluxo linear recomendado e dependências entre etapas.

Mapa do site (documentação)

  • Apresentação à diretoria — narrativa executiva detalhada; ideal para projetor ou PDF impresso.
  • Visão geral do wizard — resumo das etapas e links para cada capítulo.
  • Etapas 1 a 6 — descrição passo a passo (campos, URLs, regras) alinhada ao frontend.
  • Fluxos do usuário — macrofluxo login → dashboard → wizard.
  • Tech Stack e pastas, rotas, telas — apêndice para desenvolvimento.

Como abrir no navegador: com o frontend rodando (npm run dev no projeto Vite), acesse http://localhost:5173/documentation/index.html (ou a URL base do seu ambiente + /documentation/index.html).

A API expõe também o Swagger em /api/docs — útil para quem valida integrações, não para a apresentação de negócio.
