Sprint 0 — Fundação de engenharia (baseline)

Cadência: cada sprint encerra 14 dias após o fim do sprint anterior (S0 → 15/05, S1 → 29/05, …, S8 → 04/09/2026). Prazo alvo deste sprint: 15/05/2026. Itens abaixo: Não iniciado como trabalho formal.

Área Tarefa Critério / artefato
Repositório Política de branches, tamanho de PR, template de PR (checklist: testes, migração, breaking API) Documentado em CONTRIBUTING.md ou wiki do time
CI Pipeline em PR: npm run lint + npm run build (FE e BE separados) PR vermelho bloqueia merge em main
Config .env.example no backend e frontend com todas as chaves necessárias; validação de env no bootstrap Nest (@nestjs/config) App falha rápido se variável obrigatória ausente
Prisma Fluxo único: migrate dev local, migrate deploy em CI/CD; proibir db push em prod Script de deploy documentado; zero drift schema/prod
NestJS ValidationPipe global (whitelist + transform), filtro de exceção HTTP padronizado, tags Swagger por domínio (auth, company, wizard, …) Respostas de erro JSON consistentes; OpenAPI atualizado
Segurança Revisão CORS, rate limit (se aplicável), tamanho máximo upload Multer, sanitização de nomes de arquivo Lista de riscos + issues criadas
